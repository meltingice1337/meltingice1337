# Hey, I'm Darius 👋

Senior Full-Stack Blockchain Engineer.

I've spent the last decade writing software and the last five years going deep into blockchain. I build DeFi protocols, optimize smart contracts for gas efficiency, and design systems where security and performance are the foundation, not an afterthought. I've worked across the stack: from Solidity contracts and Rust-based infrastructure to React frontends and Node.js backends.

I also work with low-latency data pipelines for real-time blockchain indexing, confidential computing with hardware enclaves, ZK-rollup solutions, and non-custodial wallet infrastructure. My interest in blockchain goes back to university, where my bachelor thesis focused on decentralized accreditation verification using blockchain. I later pursued a master's in machine learning, with a thesis on real-time object detection for smart cities.

In my spare time I'm big into home automation, always looking for ways to make my setup smarter. Also a gamer when I get the chance.
